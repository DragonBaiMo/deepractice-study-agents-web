# 更新日志

本文件记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.1] - 2024-12-27

### 修复

- 🔧 修复内容同步脚本中 HTML 属性嵌套引号导致的 VitePress 编译错误
  - 自动转义 `<img>` 标签 `alt` 属性中的内部引号为 `&quot;`
  - 自动修复未闭合的 `<div>` 标签
- 🔧 配置 VitePress 忽略原始内容中的死链接

## [1.0.0] - 2024-12-27

### 新增

- 🎉 初始版本发布
- 📚 完整的 16 章教程内容
- 🗺️ 学习地图功能
- 📦 资源库与资源注册表
- 🔍 本地全文搜索
- 🎨 自定义主题与增强组件
  - Callout 提示块
  - Steps 步骤块
  - Demo 演示块
  - CodeRun 代码运行块
- 🔄 CI/CD 自动化流程
  - 内容规范检查
  - 链接有效性检查
  - 自动构建部署
  - 定期外链巡检
- 📱 响应式设计

### 技术栈

- VitePress 1.5.0
- Vue 3.4
- TypeScript
- Mermaid 图表支持
