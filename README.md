# Deepractice Agents 智能体开发教程

> 基于 TypeScript + AgentX + PromptX 的现代 AI 智能体开发指南

## 📚 教程简介

本教程由 **深度实践 (Deepractice)** 团队为白云学院定制开发，是一套理论与实战并重的智能体开发教程。

**技术选型：**
- **TypeScript** - 现代 AI 开发首选语言
- **AgentX** - 事件驱动的 Agent 框架
- **PromptX** - AI 角色认知系统
