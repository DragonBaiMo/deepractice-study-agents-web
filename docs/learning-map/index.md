# 学习地图

根据你的学习目标和基础，选择适合的学习路径。

## 🎯 入门路径 {#入门路径}

适合：零基础或对智能体概念不熟悉的学习者

| 阶段 | 内容 | 预计时间 |
|------|------|---------|
| 1 | [第1章 初识智能体](/docs/chapter01/) | 2小时 |
| 2 | [第2章 智能体发展史](/docs/chapter02/) | 3小时 |
| 3 | [第3章 大语言模型基础](/docs/chapter03/) | 4小时 |

完成后你将：
- 理解智能体的基本概念和工作原理
- 了解智能体技术的发展历程
- 掌握 LLM 的核心能力和边界

## 🚀 进阶路径 {#进阶路径}

适合：有一定编程基础，想深入学习智能体开发的学习者

| 阶段 | 内容 | 预计时间 |
|------|------|---------|
| 1 | [第4章 智能体经典范式](/docs/chapter04/) | 6小时 |
| 2 | [第5章 PromptX 认知系统](/docs/chapter05/) | 8小时 |
| 3 | [第6章 Deepractice 框架体系](/docs/chapter06/) | 6小时 |
| 4 | [第7章 AgentX 事件驱动框架](/docs/chapter07/) | 8小时 |

完成后你将：
- 掌握 ReAct、Plan-and-Solve、Reflection 等经典范式
- 熟练使用 PromptX 构建有认知能力的智能体
- 能够使用 AgentX 框架开发生产级应用

## 🏆 项目实战 {#项目实战}

适合：已完成进阶路径，想通过实战项目巩固技能的学习者

| 项目 | 内容 | 预计时间 |
|------|------|---------|
| 1 | [第14章 智能旅行助手](/docs/chapter14/) | 10小时 |
| 2 | [第15章 自动化深度研究](/docs/chapter15/) | 12小时 |
| 3 | [第16章 构建赛博小镇](/docs/chapter16/) | 15小时 |

完成后你将：
- 具备独立开发完整智能体应用的能力
- 掌握多智能体协作的设计模式
- 能够将智能体应用部署到生产环境
