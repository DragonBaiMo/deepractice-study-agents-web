# 第七章 智能体经典范式

在前面的章节中，我们学习了 PromptX（第四章）、AgentX（第五章）和 4P 理论（第六章），理解了 Deepractice 团队构建智能体的完整方法论。现在，让我们回归基础，深入理解智能体的经典范式。

本章将带你从零实现三种经典的智能体范式。通过手写代码，你将更深刻地理解 AgentX 框架和 PromptX 系统的设计决策：

| 范式 | 核心思想 | 适用场景 |
|------|---------|---------|
| **ReAct** | 思考与行动交替进行，边想边做 | 需要外部工具、动态调整的任务 |
| **Plan-and-Solve** | 先制定完整计划，再严格执行 | 结构清晰、步骤明确的任务 |
| **Reflection** | 执行后自我反思，迭代优化 | 对质量要求高的任务 |

通过亲手实现这些范式，你将深刻理解智能体的工作原理——只有理解了底层机制，才能在框架无法满足需求时进行定制和扩展。
